# Quran Pro — موقع قرآن كريم احترافي (مجاني 100%)

هذا مشروع موقع قرآن كريم (Static) يعمل بدون أي سيرفر خاص، ويمكن استضافته مجانًا على:
- Cloudflare Pages
- Netlify
- Vercel
- GitHub Pages

## المميزات
- قراءة القرآن بسورة/آية بتصميم حديث (RTL)
- بحث سريع بالسور + بحث داخل الآيات داخل صفحة السورة
- تشغيل التلاوة آية بآية (Mini Player) + تشغيل متتابع
- مفضلة (Bookmarks) محفوظة على جهاز المستخدم
- حفظ آخر موضع قراءة تلقائيًا
- سبحة إلكترونية عائمة + صفحة سبحة كاملة + سجل + هدف + اهتزاز
- راديو قرآن (بث مباشر) — تضع رابط Stream URL بنفسك + مفضلة محطات
- إعدادات: مظهر، حجم خط المصحف، ارتفاع السطر، التفسير/الترجمة، قارئ التلاوة
- PWA (تثبيت كتطبيق) + Service Worker (كاش لواجهة الموقع)

## تشغيل محليًا (بدون مشاكل)
> مهم: فتح `index.html` مباشرة (file://) قد يسبب مشاكل في بعض المتصفحات مع الـ fetch.
الأفضل تشغيله كسيرفر محلي:

### خيار 1: بايثون (موجود على أغلب الأجهزة)
افتح Terminal داخل مجلد المشروع ثم:
```bash
python -m http.server 8080
```
ثم افتح:
- http://localhost:8080

### خيار 2: Node (اختياري)
```bash
npx serve .
```

## نشر مجاني على Cloudflare Pages (مقترح)
1) أنشئ حساب GitHub وارفع ملفات المشروع إلى Repository.
2) افتح Cloudflare → Pages → Create a project.
3) اربط GitHub واختر الـ repo.
4) Deploy (بدون build لأن الموقع Static).

## راديو القرآن (24 ساعة)
الموقع لا يستضيف بث راديو (هذا مكلف).  
بدلًا من ذلك، تضع رابط بث مباشر (mp3/aac) لمحطة موثوقة:
- افتح صفحة **راديو القرآن** داخل الموقع وألصق الرابط.
- نصيحة: جرّب الرابط في VLC أولًا، ثم استخدمه.
